# Photos

Drop `.jpg` / `.jpeg` / `.png` / `.webp` files in this folder — they appear in the Photos app automatically (no code changes needed).

To add a title, description, date, album, or a ❤️, add an entry in `src/data/photos.ts` keyed by the exact filename (extension included). Files with no entry still show up, using the filename (minus extension) as the title.
